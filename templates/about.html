About ResumeAI

ResumeAI is an AI-powered resume screening system that helps recruiters identify the best candidates using Natural Language Processing (NLP) and Machine Learning techniques.

🔍 What is ResumeAI?

ResumeAI analyzes resumes and compares them with job descriptions using intelligent algorithms. It converts text into numerical vectors using TF-IDF and calculates similarity using cosine similarity. The system then ranks candidates based on how well their resumes match the job requirements.

🚀 Key Features

AI Resume Matching

Uses ML algorithms to match resumes with job descriptions.

PDF & DOCX Support

Automatically extracts text from resumes in multiple formats.

Match Score

Displays percentage match between resume and job role.

Skill Matching

Shows matched and missing skills clearly.

Candidate Ranking

Ranks candidates based on highest score.

Fast Processing

Analyzes multiple resumes in seconds.

⚙️ How It Works

Step 1: Upload Resume

User uploads resumes (PDF/DOCX format).

Step 2: Enter Job Description

Provide job requirements and skills.

Step 3: AI Processing

System extracts text and applies NLP techniques.

Step 4: Matching & Ranking

Candidates are scored and ranked automatically.

🧠 Technologies Used

Python • Flask • Machine Learning • NLP • TF-IDF • Cosine Similarity • Bootstrap
